Recent posts widget with date, excerpt and category filter.

== Description ==

This plugin is like the standard recent posts plugin but with more details/options. The date is diplayed in several different formats. The number of posts can be set and they can be filtered by category. The excerpt can be optionally displayed and also the amount of words in the excerpt can be set. 



== Installation ==


1. Upload `pointblank-super-recent-posts` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==



= How do i use this plugin? =

Simply go to the widgets page in the admin section and add the plugin to the section you require. Set the options as desired.
